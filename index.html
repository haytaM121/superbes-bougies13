1. إذا الموقع في جهازك (محلي)

  • إذا كنت تطور الموقع محليًا (يعني مخزّن في جهازك)، تقدر تفتحو مباشرة:

    1. افتح مجلد المشروع.

    2. دير double click على ملف index.html، أو

    3. افتح Google Chrome وروح لـ:

      pgsql
      file:///C:/path/to/your/index.html

      استبدل C:/path/to/... بمسار موقعك في جهازك.

2. إذا الموقع على Localhost

  • إذا كنت تستعمل سيرفر محلي (مثلاً XAMPP, WAMP, أو npm run dev ولا python -m http.server):

    1. شغّل السيرفر.

    2. افتح Chrome وروح لـ:

      arduino
      http://localhost:PORT

      مثلاً:

      arduino
      http://localhost:3000

3. إذا الموقع مرفوع على الإنترنت

  • بعد ما ترفع الموقع (مثلاً على GitHub Pages، Netlify، أو دومين خاص):

    1. انسخ رابط الموقع.

    2. افتحو في Google Chrome.

نصيحة للمطورين:

باش تراجع موقعك وتختبرو كيما يبان في Google:

  • كليك يمين > Inspect (أو Ctrl+Shift+I) باش تستعمل أدوات المطور.

  • تقدر تغيّر العرض لموبايل، تبين الكونسول، تشوف الأخطاء، الخ...

الخطوة 5: فتح الموقع في المتصفح

  1. بعد كتابة الأكواد، قم بحفظ الملفات (index.html و styles.css).

  2. افتح index.html في المتصفح (يمكنك فتحه مباشرة من VS Code).

    • من VS Code، اضغط على زر Right-click على index.html ثم اختر Open with Live Server (إذا قمت بتثبيت الإضافة Live Server).

    أو يمكنك فتح الملف مباشرة باستخدام المتصفح بالنقر عليه مرتين.

الخطوة 6: رفع الموقع على GitHub

  1. بعد كتابة الأكواد وتصميم الموقع، يمكنك رفع ملفات المشروع على GitHub.

  2. إذا لم تكن قد قمت بإنشاء مستودع على GitHub، يمكنك إنشاؤه بسهولة، ثم رفع الملفات.

  3. تذكر تفعيل GitHub Pages كما وضحت سابقًا، حتى يظهر الموقع على الإنترنت.
